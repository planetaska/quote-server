About

This is a simple quotes server built with Rust, Axum, and Askama.

Technology Stack

  • Rust - Programming language
  • Axum - Web framework
  • Askama - Templating engine
  • SQLite - Database for storing quotes and tags
  • SQLx - SQL toolkit for Rust
  • Tokio - Async runtime

Features

  • Browse all quotes in the database
  • View a random quote
  • RESTful API for programmatic access
  • Automatic database initialization from CSV
